# 🚑 Smart Ambulance Navigation System

## 📌 Description
This project helps ambulances reach hospitals faster by calculating the shortest and fastest route using smart algorithms.

## 💡 Features
- Real-time location tracking
- Shortest path calculation
- Map visualization using Folium
- Traffic-aware navigation (if included)

## 🛠 Technologies Used
- Python
- OpenCV
- Folium

## ▶️ How to Run
1. Install required libraries
2. Run:
   python main.py
